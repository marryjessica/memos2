# Memos 启动脚本

## 启动后端（持久化数据库）
```powershell
cd d:\CodeSpace\memos2
go run ./cmd/memos --mode dev --port 8081 --data d:\CodeSpace\memos2\data
```

## 启动前端
```powershell
cd d:\CodeSpace\memos2\web
npm run dev
```

---

## 说明
- `--data ./data`：指定数据目录为项目下的 `data` 文件夹
- 数据库文件：`d:\CodeSpace\memos2\data\memos_dev.db`
- 此数据库会**永久保存**，直到手动删除

## 如何删除数据库
如果需要清空所有数据，删除 `data` 文件夹即可：
```powershell
Remove-Item -Recurse -Force ./data
mkdir data
```
