# 修复登录问题 - 清理步骤

## 方法1：在浏览器开发者工具清理（推荐）
1. 打开 http://localhost:3001
2. 按 F12 打开开发者工具
3. 进入 "Application" 或 "应用" 标签页
4. 左侧找到 "Storage" / "存储"
5. 点击 "Clear site data" / "清除网站数据"
6. 刷新页面（F5）

## 方法2：使用隐身窗口
1. Ctrl+Shift+N 打开Chrome隐身窗口
2. 访问 http://localhost:3001/auth/signup
3. 注册新用户

## 方法3：清除浏览器所有数据
1. Chrome设置 -> 隐私和安全 -> 清除浏览数据
2. 选择"Cookie和其他网站数据"
3. 清除

---

**清理后的操作**：
1. 访问 http://localhost:3001/auth/signup
2. 注册用户（用户名：admin，密码：admin12345）
3. 注册成功后会自动登录
